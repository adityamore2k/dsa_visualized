Searching algorithms

Linear search

Consider array of elements: arr={A,B,C,D,E,F,G,H}. The algorithm for it is:

  1. Read search element
  2. Set integer i=0 and repeat steps 3 to 4 till i reaches end of array
  3. Match key with arr[i]
  4. If any key matches, return index. Else, increment by 1;
